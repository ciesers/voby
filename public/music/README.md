# Music Files

Please add your kawaii techno music file here:

- `kawaii-techno.mp3` - Main background music file
- `kawaii-techno.ogg` - Optional OGG format for better browser compatibility

The music will play automatically when the page loads and loop continuously while the spinning dog animation plays.

You can find free kawaii techno music on:
- YouTube (search "kawaii techno" or "kawaii future bass")
- Free music sites like Freesound.org
- Or create your own!

Make sure you have the rights to use any music files you add.
