# Sound Files

Please add your bark sound file here:

- `bark.mp3` - Dog bark sound effect
- `bark.ogg` - Optional OGG format for better browser compatibility

The website will use a fallback beep sound if the bark file is not found, but adding a real bark sound will make it much better!

You can find free dog bark sounds on:
- Freesound.org
- Zapsplat.com
- Or record your own!
